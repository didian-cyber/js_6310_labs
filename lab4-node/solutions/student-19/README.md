# BeautyBooker Bot 

# Описание
Telegram бот для записи в салон красоты. Позволяет пользователям выбирать услуги, мастеров, дату и время для записи.

# Функциональность
Основные команды:

/start - Начало работы с ботом
/book - Запись на услугу (полный процесс бронирования)
/services - Просмотр всех доступных услуг и цен
/history - История посещений и персональные акции

## Установка и запуск
1. Клонируйте репозиторий
2. Установите зависимости:
- npm install
3. Создайте файл .env в корне проекта:
- BOT_TOKEN=your_telegram_bot_token_here
4. Запустите Telegram бот:
- npm run dev
5. Запустите тесты
- npm test - Обычные тесты
- npm run test:coverage - Тесты с покрытием
- npm run lint - Проверка кода
